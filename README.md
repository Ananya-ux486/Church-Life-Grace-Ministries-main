# Life Grace Ministries

Church website built with **HTML**, **CSS**, **Bootstrap 5**, and vanilla JavaScript.

## How to Run

Double-click `index.html` to open the site in your browser. No install or server required — this project uses only HTML, CSS, Bootstrap, and JavaScript.

## Project Structure

```
index.html          # Main page (all sections)
css/style.css       # Custom styles (same design as original)
js/main.js          # Interactivity (menu, slider, modal, etc.)
assets/             # Images and videos
public/             # Favicon
```

## Features

- Responsive layout with Bootstrap 5 grid
- Hero image slider
- Photo gallery (Swiper)
- Scroll animations (AOS)
- Donation modal
- Lord's City video player
- Contact form
